Harder, better, stronger!

Bon, il est temps maintenant de sortir de google chrome, et de créer/modifier directement vos fichiers HTML et CSS...



Ouvre le dossier petit_bain_web qui se trouve sur le Bureau de ton poste de travail.
Tu y trouveras les fichiers 19.html et 19.css, et quelques images en plus.

Ouvre ces fichiers 19.html et 19.css avec Sublim Text, en faisant un click droit sur les fichiers et en clickant sur Sublim Text dans le sous menu "open with" dans les applications.
(Sublim Text est un éditeur de texte, comme Word ou Open Office, mais spécifiquement pour le code. Il en existe d'autres, mais pourquoi pas lui ?)


Tu peux maintenant écrire du code html dans 19.html, à l'endroit indiqué, et du css dans 19.css
Pour te faciliter la tâche si tu veux inclure des images, nous t'avons déjà fourni quelques unes.


Pour visualiser votre travail, rien de plus simple : double-clique sur 19.html, le fichier va s'ouvrir dans un navigateur web.
À chaque fois que vous modifiez l'un des fichiers 19.html ou 19.css, tu n'auras qu'à rafraichir la page dans le navigateur.


Une astuce : sur ta page web dans le navigateur : click droit, puis 'Inspect'
La console du navigateur s'ouvrira.
Et tu pourras y voir tout ton code et le modifier !
Tu verras en direct les modifications sur votre page, ce qui est très pratique pour travailler.
Mais attention, ces modifications sont provisoires et disparaîtront au prochain rafraichissement de page.
N'oublie pas de copier-coller ces modifications provisoires dans tes fichiers 19.html et 19.css pour les enregister.


Tu as carte blanche, sois créatif.ve, fais-toi plaisir !
