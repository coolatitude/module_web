page 5:

Harder, better, stronger!
Bon, il est temps maintenant de sortir de google chrome, et de créer/de modifier directement tes fichiers HTML et CSS...



Ouvre le dossier petit_bain_web qui se trouve sur le Desk de ton poste de travail.
Tu y trouveras les fichiers 19.html et 19.css, et quelques images en plus.

Ouvre ces fichiers 19.html et 19.css avec Sublim Text, en glissant le dossier petit_bain_web sur cette icône dans les applications.
(Sublim Text est un éditeur de texte, comme Word ou Open Office, mais spécifique pour le code. Il en existe d'autres, mais pourquoi pas lui ?)


Tu peux maintenant écrire du code html en 19.html, à l'endroit indiqué, et du css en 19.css
Pour te faciliter la tâche si tu veux inclure des images, nous t'en avons déjà fourni quelques-unes.


Pour visualiser ton travail, rien de plus simple : double-clique sur 19.html, le fichier va s'ouvrir dans un navigateur web.
À chaque fois que tu modifies l'un des fichiers 19.html ou 19.css, tu n'auras qu'à rafraîchir la page dans le navigateur.


Une astuce : sur ta page web dans le navigateur : clic droit, puis 'Inspect'
La console du navigateur s'ouvre.
Et tu pourras y voir tout ton code et le modifier !
Tu verras en direct les modifications sur ta page, ce qui est très pratique pour travailler.
Mais attention, ces modifications sont provisoires et disparaîtront au prochain rafraîchissement de page.
N'oublie pas de copier-coller ces modifications provisoires dans tes fichiers 19.html et 19.css pour les enregister.




Tu as carte blanche, sois créatif.ve, fais-toi plaisir !
