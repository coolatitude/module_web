*****************************************************************
                  PETIT BAIN: TUTORS INFORMATION
*****************************************************************

Welcome to the Petit Bain, a coding initiation module created by 19 students

This project has 3 internal modules: C, Shell, Web

HOW TO INSTALL:

1. Type command "cd ~/petit_bain && sh install.sh"
2. Close terminal window and open a new one.
***** IMPORTANT: This module wont work in the same window where it was installed.

HOW TO LAUNCH / PLAY:

1. Open a new terminal window (fullscreen is highly recommended)
2. Type command "pb_fr" for french, "pb_en" for english, or "pb_nl" for Dutch.
3. Enjoy :)
